# GorjVPN-panel
GorjVPN یک راه‌حل مدرن و سبک برای دور زدن محدودیت‌های اینترنتی است که هیچ سرور مجازی (VPS) نیاز ندارد. کنترل اکانت‌ها، اعتبارسنجی دستگاه و تاریخ انقضا از طریق یک فایل رمزنگاری‌شده روی GitHub انجام می‌شود و ترافیک واقعی با استفاده از جعل SNI و Google Apps Script به عنوان رله عبور می‌کند. و همچنین پنل محور است.

# هسته
تمام هسته عبور از فیلترینگ متعلق به  https://github.com/therealaleph/MasterHttpRelayVPN-RUST که خود نسخه ای از https://github.com/masterking32/MasterHttpRelayVPN است

# این اسکریپت به صورت گرافیکی و ایمن می تواند یک script id را در چند کاربر پخش کند و ادمین بتواند دترسی هارا مشاهده یا محدود یا اضفه کند

# ساخت ID اختصاصی برای هر دستگاه
```
python GorjVPN_ID.py
```

# پنل کنترل کاربران 
```
python admincontrol.py
```

# اجرا روی کاربران
نکته: بهتر است خروجی exe به افراد داده شود
```
python GorjVPN.py
```

# همچنین از لینک یک repositorie گیت هاب به عنوان سرور واسط بین ادمین و کاربران به صورت کاملا رمزنگاری شده استفاده می کند
